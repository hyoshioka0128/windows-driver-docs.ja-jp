# <a name="oid_gen_co_rcv_crc_error"></a>OID_GEN_CO_RCV_CRC_ERROR

OID_GEN_CO_RCV_CRC_ERROR OID は、巡回冗長検査 (CRC) エラーで受信した Pdu の数を指定します。

## <a name="requirements"></a>要件

**バージョン**: Windows Vista 以降の**ヘッダー**: Ntddndis (Ndis .h を含む)
