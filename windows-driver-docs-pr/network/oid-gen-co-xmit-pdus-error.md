# <a name="oid_gen_co_xmit_pdus_error"></a>OID_GEN_CO_XMIT_PDUS_ERROR

OID_GEN_CO_XMIT_PDUS_ERROR OID は、NIC が送信に失敗した Pdu の数を指定します。

## <a name="requirements"></a>要件

**バージョン**: Windows Vista 以降の**ヘッダー**: Ntddndis (Ndis .h を含む)
