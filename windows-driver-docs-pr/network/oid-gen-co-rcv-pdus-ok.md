# <a name="oid_gen_co_rcv_pdus_ok"></a>OID_GEN_CO_RCV_PDUS_OK

OID_GEN_CO_RCV_PDUS_OK OID は、NIC が受信した Pdu の数をエラーなしで指定し、バインドされたプロトコルを示します。

## <a name="requirements"></a>要件

**バージョン**: Windows Vista 以降の**ヘッダー**: Ntddndis (Ndis .h を含む)
