# <a name="oid_gen_co_bytes_xmit"></a>OID_GEN_CO_BYTES_XMIT

OID_GEN_CO_BYTES_XMIT OID は、エラーなしで送信される Pdu のバイト数を指定します。

## <a name="requirements"></a>要件

**バージョン**: Windows Vista 以降の**ヘッダー**: Ntddndis (Ndis .h を含む)
