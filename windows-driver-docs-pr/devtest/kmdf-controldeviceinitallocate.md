# <a name="controldeviceinitallocate-rule-kmdf"></a>ControlDeviceInitAllocate ルール (kmdf)


**Controldeviceinitallocate**ルールは、コントロールデバイスオブジェクトに対して、ドライバーが[**WdfDeviceCreate**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdevicecreate)を呼び出す前に、フレームワークデバイスオブジェクト初期化メソッド[**Wdfcontroldeviceinitallocate**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfcontrol/nf-wdfcontrol-wdfcontroldeviceinitallocate)を呼び出す必要があることを指定します。

|              |      |
|--------------|------|
| ドライバー モデル | KMDF |
