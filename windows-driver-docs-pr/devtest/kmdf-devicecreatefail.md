# <a name="devicecreatefail-rule-kmdf"></a>DeviceCreateFail ルール (kmdf)


**DeviceCreateFail**ルールは、 \_ \_ \_ \_ [**WdfDeviceCreate**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdevicecreate)への呼び出しが失敗した場合に、.evt WDF DRIVER DEVICE ADD がエラー状態を返すことを指定します。

ドライバーがデバイスを処理するには、デバイスオブジェクトが正常に作成されている必要があります。

**ドライバーモデル: KMDF**
