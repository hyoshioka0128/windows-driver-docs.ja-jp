# <a name="storportdeprecated-rule-storport"></a>StorPortDeprecated ルール (storport)


このルールは、この非推奨のルーチン ( **StorPortValidateRange**または**Storportlogerror**) のいずれかがドライバーによって呼び出されないことを確認します。

**ドライバーモデル: Storport**
