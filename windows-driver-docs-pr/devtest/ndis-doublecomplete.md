# <a name="doublecomplete-rule-ndis"></a>DoubleComplete rule (ndis)


DoubleComplete 規則は、NDIS ドライバーがオブジェクト識別子 (OID) 要求を複数回完了できないことを指定します。

このルールは、 **Miniportoidrequest**コールバック関数が NDIS\_STATUS\_SUCCESS を返したときに、その要求に対して**NdisMOidRequestComplete**関数を呼び出すことができないことを確認します。 また、このルールでは、 **Miniportoidrequest**が pending status を返したときに、ドライバーが**NdisMOidRequestComplete**関数をその要求に対して複数回呼び出すことができないように指定しています。

|              |      |
|--------------|------|
| ドライバー モデル | NDIS |
