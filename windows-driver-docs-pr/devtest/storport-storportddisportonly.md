# <a name="storportddisportonly-rule-storport"></a>StorPortDDIsPortOnly ルール (storport)


このルールには、storport ミニポートでは呼び出さない、StorPort ポートのみの DDIs (インタロックされた関数を除く) の一覧が含まれています。

**ドライバーモデル: Storport**
