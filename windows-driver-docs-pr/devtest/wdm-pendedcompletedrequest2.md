# <a name="pendedcompletedrequest2-rule-wdm"></a>PendedCompletedRequest2 ルール (wdm)


**PendedCompletedRequest2**規則は、ディスパッチルーチンが保留中の IRP を完了できるため、 [**IoCallDriver**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-iocalldriver)または[**pocalldriver**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-pocalldriver)を呼び出した後に待機が必要であることを指定します。

**ドライバーモデル: WDM**
