# <a name="removelockcheck-rule-wdm"></a>RemoveLockCheck 規則 (wdm)


**Removelockcheck**規則は、 [**IoAcquireRemoveLock**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-ioacquireremovelock) [**IoReleaseRemoveLockAndWait**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-ioreleaseremovelockandwait) \_ \_ minorfunction irp IoAcquireRemoveLock REMOVE DEVICE で IRP MJ PNP を処理するときに、と IoReleaseRemoveLockAndWait の呼び出しが正しく使用されることを検証し \_ \_ \_ ます。

**ドライバーモデル: WDM**
