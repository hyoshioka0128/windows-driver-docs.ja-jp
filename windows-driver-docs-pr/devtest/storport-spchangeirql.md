# <a name="spchangeirql-rule-storport"></a>SpChangeIrql ルール (storport)


このルールは、StorPort コールバックルーチンが、呼び出されたレベルと同じ IRQL レベルで返されることを確認します。

Storport ミニポートエントリポイントまたはコールバックルーチン内の IRQL レベルを変更するすべての DDI に適用されます。

**ドライバーモデル: Storport**
