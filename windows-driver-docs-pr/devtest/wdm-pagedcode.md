# <a name="pagedcode-rule-wdm"></a>PagedCode ルール (wdm)


**PagedCode**規則は、 **IRQL &lt; = APC \_ レベル**で実行されている場合にのみ、ドライバーが[**ページング \_ コード**](https://docs.microsoft.com/windows-hardware/drivers/kernel/mm-bad-pointer)マクロを呼び出すことを指定します。

**ドライバーモデル: WDM**
