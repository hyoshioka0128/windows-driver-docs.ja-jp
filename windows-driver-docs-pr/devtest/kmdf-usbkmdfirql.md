# <a name="usbkmdfirql-rule-kmdf"></a>Usbkmdファイヤ Ql ルール (kmdf)


**Usbkmdql ql**ルールは、kmdf ドライバーが、正しくない IRQL レベルで USB 固有のデバイスドライバーインターフェイス (DDI) を呼び出さないことを指定します。

**ドライバーモデル: KMDF**
