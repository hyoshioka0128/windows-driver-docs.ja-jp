# <a name="iobuildfsdfree-rule-wdm"></a>IoBuildFsdFree ルール (wdm)


**IoBuildFsdFree**規則は、以前に[**IoBuildAsynchronousFsdRequest**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-iobuildasynchronousfsdrequest)で割り当てられた irp でのみ、ドライバーが[**iofreeirp**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-iofreeirp)を使用するように指定します。

**ドライバーモデル: WDM**
