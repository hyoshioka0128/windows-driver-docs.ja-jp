# <a name="syncreqsend2-rule-kmdf"></a>SyncReqSend2 ルール (kmdf)


**SyncReqSend2**ルールは、同期要求の送信に0以外のタイムアウト値が設定されていることを指定します。

要求オプションで有効なタイムアウトを設定せずに、ドライバーが[**Wdfrequestsend**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfrequest/nf-wdfrequest-wdfrequestsend)を呼び出すと、ハードウェアがすぐに応答しない場合にスレッドが停止する可能性があります。

**ドライバーモデル: KMDF**
