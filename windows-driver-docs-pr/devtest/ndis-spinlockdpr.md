# <a name="spinlockdpr-rule-ndis"></a>SpinLockDpr rule (ndis)


**SpinLockDpr**規則は、NDIS スピンロックインターフェイスの正しい使用を検証します。

このルールは、スピンロックがロック解除された状態の場合にのみ、 [**NdisDprAcquireSpinLock**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndis/nf-ndis-ndisdpracquirespinlock)の呼び出しが行われるように指定します。 また、このルールは、ミニポートハンドラールーチンが終了する前にスピンロックが解放されたことを確認します。

**ドライバーモデル: NDIS**
