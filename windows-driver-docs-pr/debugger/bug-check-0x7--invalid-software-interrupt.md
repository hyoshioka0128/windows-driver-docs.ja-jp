# <a name="bug-check-0x7-invalid_software_interrupt"></a>バグチェック 0x7: 無効な \_ ソフトウェア \_ 割り込み


無効な \_ ソフトウェアの割り込みのバグチェックには、 \_ 0x00000007 の値が含まれています。

このバグチェックは非常に頻繁に行われません。
