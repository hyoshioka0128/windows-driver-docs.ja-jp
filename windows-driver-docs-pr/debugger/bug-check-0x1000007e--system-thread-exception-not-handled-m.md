# <a name="bug-check-0x1000007e-system_thread_exception_not_handled_m"></a>バグチェック 0x1000007E: システム \_ スレッド \_ 例外が \_ 処理されませんでした \_ \_ M


システム \_ スレッドの \_ 例外が \_ 処理されませんでした \_ \_ M バグチェックの値は0x1000007e です。 これは、エラーハンドラーがキャッチしなかった例外がシステムスレッドによって生成されたことを示します。

バグチェック0x1000007E には、[**バグチェック 0x7e**](bug-check-0x7e--system-thread-exception-not-handled.md) (システム \_ スレッド \_ 例外が \_ 処理されていません) と同じ意味とパラメーターがあり \_ ます。

> [!IMPORTANT]
> このトピックはプログラマーを対象としています。 コンピューターの使用中にブルースクリーンのエラーコードが表示された顧客の場合は、「[ブルースクリーンエラーのトラブルシューティング](https://www.windows.com/stopcode)」を参照してください。
