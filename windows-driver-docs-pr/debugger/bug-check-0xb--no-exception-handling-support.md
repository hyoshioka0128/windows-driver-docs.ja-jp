# <a name="bug-check-0xb-no_exception_handling_support"></a>バグチェック 0xB: \_ 例外 \_ 処理は \_ サポートしていません


例外処理がサポートされていないことを示す \_ \_ \_ バグチェックには、0x0000000B の値が含まれています。

このバグチェックは非常に頻繁に行われません。
