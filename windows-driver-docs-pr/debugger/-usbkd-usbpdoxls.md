# <a name="usbkdusbpdoxls"></a>!usbkd.usbpdoxls


**! Usbkd. usbpdoxls**コマンドは、USB ハブドライバーによって作成されたすべての物理デバイスオブジェクト (pdos) に関する情報を表示します。 このコマンドは、PDO ごとに、コンマ区切り値の行を表示します。

```dbgcmd
!usbkd.usbpdoxls
```
