# <a name="bug-check-0x3d-interrupt_exception_not_handled"></a>バグチェック 0x3D: 割り込み \_ 例外が \_ 処理されていません \_


割り込み \_ 例外が \_ ハンドルされていないバグチェックには、 \_ 値0x0000003d があります。

このバグチェックは非常に頻繁に行われません。
