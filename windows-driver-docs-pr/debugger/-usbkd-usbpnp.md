# <a name="usbkdusbpnp"></a>!usbkd.usbpnp


**! Usbkd. usbkd**コマンドは、USB ハブに関する状態のコンテキスト情報を表示します。

```dbgcmd
!usbkd.usbpnp DeviceExtension
```
